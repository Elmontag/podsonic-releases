Features

Everything iTunes should have been.

Manage your music collection, sync with Subsonic servers, keep your devices up to date, and enjoy music in focus — all from one app.

Music Library

A powerful local library with instant search, non-destructive tag editing and automatic metadata enrichment from online sources.

  • Edit ID3 tags with a non-destructive overlay model
  • Cover art from TheAudioDB & Last.fm
  • Instant full-text search via SQLite FTS5 — even with tens of thousands of tracks
  • FLAC, MP3, AAC, ALAC and more
  • Multiple source folders — files stay where they are
  • Rip audio CDs with automatic metadata via MusicBrainz
  • File & folder import with configurable target structure

Subsonic Sync

Connect to any Subsonic-compatible server and selectively sync music to your local library.

  • Navidrome, Airsonic, Gonic and all Subsonic-compatible servers
  • Selective sync per artist, album or playlist
  • Live progress with real-time WebSocket updates
  • Network-safe: atomic writes protect NAS mounts
  • Background sync — keep using the app while it downloads

Device Sync

Transfer your library to portable devices, external drives and iPods.

  • Sync to local folders and mounted drives
  • iPod Classic support (see compatibility below)
  • Cover art transferred directly to the iPod
  • FAT32-compatible on Windows, macOS and Linux

Playback

A focused listening experience — from the full-screen player to synchronized lyrics.

  • Full-screen player with album art
  • Queue management — reorder, shuffle, save as playlist
  • Synchronized lyrics via LRCLib
  • Play from any library view

Apple Devices

iPod support — seriously.

Podsonic can sync your music library directly to an iPod Classic using the native iTunes database format.

Apple devices are supported as a sync target for typical use cases.
The following table shows the current test status for iPod models.
Note: iTunes detects when the database has been written by a different app and may prompt to re-sync — this is a known limitation.

Device Status Notes
iPod Classic 6th Gen Tested Fully tested and working
iPod Classic 5th Gen Likely works Same format, not tested
iPod Classic 7th Gen Likely works Same format, not tested
iPod Classic (other generations) Untested Unknown compatibility
iPod Nano (all generations) Likely not working Different firmware / protocol
iPod Shuffle (all generations) Likely not working Different firmware / protocol
iPod Touch (all generations) Likely not working iOS-based, different protocol
